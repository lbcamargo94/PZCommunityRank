description=[h1]EN - PZ Community Rank[/h1]
description=
description=Turn every death into a recorded achievement. [b]PZ Community Rank[/b] is the official mod of [b]Brasileirão PZ[/b] — the Brazilian Project Zomboid survival championship.
description=
description=[h2]How It Works[/h2]
description=
description=When you die, the mod automatically generates a [b]PZR code[/b] with all your survival data. Submit it through the [b]PZ-Rank Companion[/b] (external app) to register your result in the official ranking.
description=
description=Your data is also synced automatically during the run:
description=[list]
description=[*] Every ~5 real-time minutes
description=[*] On any skill level up
description=[*] Every 5 zombie kills
description=[*] Every in-game day
description=[*] On save or world exit
description=[/list]
description=
description=[h2]BRASILEIRAO PZ Game Mode[/h2]
description=
description=The mod adds the [b]BRASILEIRAO PZ[/b] mode directly to the new game screen. Selecting it automatically applies the official championship sandbox — no manual configuration needed.
description=
description=[h2]Anti-Cheat[/h2]
description=
description=The mod checks and corrects the sandbox every ~5 minutes. If any preset value is changed during a run, the run is [b]permanently disqualified[/b] — the generated code is flagged as invalid in the ranking.
description=
description=[h2]Commands[/h2]
description=
description=[list]
description=[*] [b]/rank[/b] in chat — generate the code manually at any time
description=[*] [b]Right-click world → "Gerar Rank"[/b] — alternative via context menu
description=[/list]
description=
description=[h2]Compatibility[/h2]
description=
description=[list]
description=[*] [b]Build 42.19+[/b] — Single Player and Multiplayer
description=[*] In MP, each player generates and manages their own code independently
description=[*] No conflicts with other mods
description=[/list]
